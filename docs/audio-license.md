# Public showcase audio

The GitHub Pages showcase ships with one full-length jazz track:

- Title: `Jazz at the park`
- Artist / uploader: `Manwithmetalpig`
- Source page: <https://commons.wikimedia.org/wiki/File:Jazz_at_the_park.ogg>
- Direct file URL: <https://upload.wikimedia.org/wikipedia/commons/0/03/Jazz_at_the_park.ogg>
- License: `CC0 1.0` / Creative Commons Zero, Public Domain Dedication
- License deed: <https://creativecommons.org/publicdomain/zero/1.0/>
- Description from the source page: flute and tenor sax jazz duo at a public park

Why this file is used:

- it is a real jazz recording
- it can be redistributed publicly inside the repository
- it works well inside GitHub Pages because the file is lightweight enough for a static showcase

The file is stored in `docs/assets/audio/jazz-at-the-park.ogg` and is only used by the static public showcase. The full Docker application still uses the project demo catalog and local development assets under `apps/web/public` and `storage/`.
