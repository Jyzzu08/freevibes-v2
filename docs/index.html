Architecture

Built to be portfolio-ready today and deployable tomorrow

Frontend

Next.js App Router, Zustand player state, Tailwind styling and a FreeVibes visual system centered on black, yellow and white.

Backend

NestJS modules for auth, catalog, playlists, library, playback, uploads and admin, with DTO validation, guards and Swagger.

Data

Prisma over PostgreSQL, Redis ready for infra growth, and a bundled legacy XML dataset so the repository can seed itself without external folders.

Hosting strategy

GitHub Pages serves this public showcase. The full stack remains Docker-driven and ready for a future split deploy to managed backend and database services.
