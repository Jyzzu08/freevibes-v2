# Publication Kit

## GitHub short description

FreeVibes 2.0 is a Docker-first music streaming platform built with Next.js, NestJS, Prisma, PostgreSQL and Redis.

## GitHub long description

FreeVibes 2.0 is a portfolio-grade full-stack streaming platform that evolves my original FreeVibes project into a modern product. It combines a Next.js frontend, a NestJS API, Prisma, PostgreSQL, Redis, Docker Compose, real playlists, user library flows, uploads, playback history and an admin panel, while preserving the dark black-yellow-white visual identity of the original concept.

## Portfolio summary

FreeVibes 2.0 is a self-directed full-stack product case study focused on product thinking, backend architecture, Docker-first development and real user flows. The project demonstrates how I turn an initial front-end concept into a modular platform with authentication, persistence, admin tooling, media handling and a publishable developer experience.

## Technical highlights

- Monorepo with Next.js 15, NestJS and shared packages
- Docker Compose setup with web, api, postgres, redis and nginx
- Prisma data model for users, profiles, artists, albums, tracks, playlists and library state
- JWT auth with refresh tokens, guards and route hardening
- Real playlist CRUD, favorites, saved albums, followed artists and playback history
- Local uploads architecture prepared for future S3 migration
- Public GitHub Pages showcase with a full-length CC0 jazz track plus full-stack Docker demo workflow
